# Telegram Publisher Bot

Система автоматической публикации постов в Telegram канал с планировщиком и deep linking.

## 📋 Функциональность

- **Управление статьями**: добавление, просмотр, удаление, редактирование
- **Автоматическая публикация**: тизер и пост-повтор из одной статьи
- **Deep linking**: доступ к полному тексту по ссылке
- **Планировщик**: независимый процесс для публикации по расписанию
- **Статистика**: мониторинг работы системы
